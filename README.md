# Пример настройки CI/CD и юнит-тестов

## Тестовые классы

### Point
 - Конструктор по двум числам
 - Вычисление расстояния между точками
 - Вычисление центра отрезка по двум точкам

### Line
 - Конструкторы по двум точкам и коэффициентам
 - Функция принадлежности точки к прямой
 - Вычисление абсцисы и ординаты
